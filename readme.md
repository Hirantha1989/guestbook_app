# Guest Book Application
Simple comments moderation application, which is built using Laravel 5.8 framework.

### Requirements
    Laravel >=5.8
    PHP >= 7.0
    
## Features
- Add Comments/ messages
- Moderate messages
 

## Installation

1. Run
    ```
    composer install
    ```
2. Run the migrations:  

    ```
    php artisan migrate
    ```
